Este repositorio contiene la actividad del Modulo 1 de Programacion 1. El programa esta hecho en C++ y cubre los temas principales que vimos en clase.

## Que contiene el programa

| Funcion | Que hace |
|---|---|
| `saludar_usuario()` | Saludo al iniciar el programa |
| `mostrar_tipos_datos()` | int, float, char, bool, double con setw |
| `explicar_palabras_reservadas()` | Palabras clave y reglas de nombrado |
| `ejemplo_uso_bibliotecas()` | Funcion propia + bibliotecas estandar |
| `manejo_entrada_salida()` | cin, cout con manipuladores de flujo |
| `estructurar_programa()` | Modularidad y estructura de C++ |
| `ejemplo_funciones_basicas()` | Funciones con parametros y retorno |
| `calculate_factorial()` | Funcion recursiva - Bonus ⭐ |
| `leer_escribir_archivo()` | Leer y escribir con fstream |
| `recorrer_vectores_matrices()` | Vectores y matrices con for |

## Como correrlo

```bash
g++ tarea_modulo1.cpp -o tarea
./tarea
```
